My playground, and passion project. Learnt something new at work and want to try it out? I test it here.
It's gone through 2 web frameworks, 3 hosts and countless "what if I rewrote it like that" dating back to 2021.

While it will never be "done", I've now built it in a way where it should be able to maintain itself for when I do eventually lose interest in this.

Most recently, after [a number of platform improvements](/blog/2026-pokecompanion-rewrite) I started working on a proper mobile app as well! I'm currently looking for testers on Android, please do sign up on [pokecompanion.com/app/invite](https://pokecompanion.com/app/invite?utm_source=portfolio&utm_campaign=projects).

Frontend: Svelte Kit deployed to Cloudflare. React Native via Expo soon 👀

Backend: Shared ExpressJS API on a VPS. Pocketbase acting as a user database, and Redis for caching to the PokeAPI.
